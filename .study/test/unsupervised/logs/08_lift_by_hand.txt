[직관 1] 우유 → 요거트  (거래 N = 20 )
  P(우유)        = 0.50   (우유가 든 거래 비율)
  P(요거트)      = 0.20   (요거트가 든 거래 비율)
  기대 동시구매  = P(우유)*P(요거트) = 0.50*0.20 = 0.10  (= 10%)
  실제 동시구매  = P(우유∩요거트)     = 0.15  (= 15%)
  confidence     = P(우유∩요거트)/P(우유) = 0.15/0.50 = 0.30
  lift           = 실제/기대 = 0.15/0.10 = 1.50
  → "상관 없다면 10%만 같이 사야 하는데 15%나 샀다" = 기대보다 1.5배.
  부호 점검: 기호는 P(A∪B)로 쓰지만(두 상품을 한 바구니 set으로 합침),
            실제로 세는 것은 둘을 '동시에' 산 거래수 = 교집합 P(A∩B).

[직관 2] 만두 → 굴소스  (거래 N = 96 )
  P(만두)   = 0.4167
  P(굴소스) = 0.0417   (단독으로는 거의 안 팔리는 희귀 상품)
  confidence = P(만두∩굴소스)/P(만두) = 0.0417/0.4167 = 0.1000  (≈10%)
  lift = confidence / P(굴소스) = 0.1000/0.0417 = 2.40
  → "조건 만족한 사람 중 10%만 샀는데 왜 의미?" : 분모 P(굴소스)=0.042가
     워낙 작아, 낮은 confidence라도 lift는 2.4배로 커진다.
