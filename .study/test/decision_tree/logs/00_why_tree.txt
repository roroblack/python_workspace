# 00_why_tree
============================================================
XOR 배치 — 한 직선으로는 0/1 을 가를 수 없는 데이터:
  x1=0, x2=0  → 클래스 0
  x1=0, x2=1  → 클래스 1
  x1=1, x2=0  → 클래스 1
  x1=1, x2=1  → 클래스 0

결정트리 학습 정확도: 1.0000  (XOR 도 완벽 분리)
트리 깊이(depth): 2,  잎(leaf) 개수: 4

트리가 만든 if-else 규칙:
|--- x1 <= 0.50
|   |--- x2 <= 0.50
|   |   |--- class: 0
|   |--- x2 >  0.50
|   |   |--- class: 1
|--- x1 >  0.50
|   |--- x2 <= 0.50
|   |   |--- class: 1
|   |--- x2 >  0.50
|   |   |--- class: 0
