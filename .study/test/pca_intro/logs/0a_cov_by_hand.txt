원본 X (행=샘플 3, 열=변수 2):
[[2. 3.]
 [4. 5.]
 [6. 7.]]

Step1 변수 평균 μ = [4. 5.]  (열 방향 평균: 변수1, 변수2)
Step2 중심화 Xc = X - μ:
[[-2. -2.]
 [ 0.  0.]
 [ 2.  2.]]

Step3 shape 추적: Xc.T(2, 3) @ Xc(3, 2) -> (d×d)=(2, 2)
  → n(샘플 수)이 곱셈 안에서 사라지고 변수 개수 d×d 칸만 남는다.
  C[0][0] = Σ(Xc[:,0]·Xc[:,0]) / (n-1) = 8.0/2 = 4.0000
  C[0][1] = Σ(Xc[:,0]·Xc[:,1]) / (n-1) = 8.0/2 = 4.0000
  C[1][0] = Σ(Xc[:,1]·Xc[:,0]) / (n-1) = 8.0/2 = 4.0000
  C[1][1] = Σ(Xc[:,1]·Xc[:,1]) / (n-1) = 8.0/2 = 4.0000

손 계산 공분산 행렬 C_hand:
[[4. 4.]
 [4. 4.]]

행렬식 (Xc.T @ Xc)/(n-1):
[[4. 4.]
 [4. 4.]]

np.cov(X, rowvar=False):
[[4. 4.]
 [4. 4.]]

손 계산 == 행렬식 == np.cov ? True

대각선 C[0][0]=4.0(변수1 분산), C[1][1]=4.0(변수2 분산)
비대각선 C[0][1]=4.0 = C[1][0]=4.0  (두 변수의 관계, 대칭)
비대각선이 양수이고 분산과 같은 크기 → 두 변수가 거의 완전히 같이 커진다.
