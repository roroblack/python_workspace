# PCA

## 한 줄 정의

PCA는 고차원 데이터를 정보 손실을 감수하면서 더 적은 축으로 압축하는 차원 축소 방법이다.

## 왜 배웠나

차원이 많아지면 거리 기반 모델이 불안정해지고, 시각화와 해석도 어려워진다.

## 핵심 질문

- PCA는 무엇을 기준으로 새 축을 고르는가?
- PCA 전에 표준화가 필요한 이유는 무엇인가?
- 설명분산비와 누적 설명분산비는 어떻게 읽는가?
- 차원 축소 후 성능이 좋아지거나 나빠지는 이유는 무엇인가?

## 직접 검증한 것

- digits 데이터에서 설명분산비 기준으로 차원 수를 선택했다.
- PCA + KNN 조합에서 차원 수에 따른 정확도 변화를 확인했다.
- 복원 MSE를 통해 차원 축소의 손실을 확인했다.

## 헷갈리는 지점

- PCA는 지도학습 모델이 아니라 변환 방법이다.
- PCA는 분류 성능을 직접 최적화하지 않는다.
- 스케일이 큰 특성이 주성분을 지배할 수 있으므로 표준화가 중요하다.

## 연결 개념

- 차원의 저주
- 공분산
- 표준화
- KNN
- t-SNE
- 설명분산비
