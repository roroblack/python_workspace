# 수소차 통합 대시보드 프로젝트

## 한 줄 요약

수소차 관련 데이터를 수집하고 DB에 저장한 뒤, 대시보드 형태로 통합해 보여준 첫 단위 프로젝트다.

## 연결 분야

- DB 설계
- SQL
- 크롤링
- 데이터 수집
- ERD
- 팀 협업
- 발표 자료 작성

## 다시 볼 질문

- 어떤 데이터를 수집했고, 왜 그 데이터가 필요했는가?
- 수집한 데이터를 어떤 테이블 구조로 저장했는가?
- PK와 FK는 어떤 기준으로 잡았는가?
- 크롤링, DB, 화면 표시 흐름은 어떻게 연결되었는가?
- 프로젝트에서 가장 큰 병목은 데이터 수집, 모델링, 구현 중 무엇이었는가?
